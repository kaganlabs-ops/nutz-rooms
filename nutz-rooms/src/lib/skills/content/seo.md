# SEO Basics

## SEO TAKES TIME

seo = slow burn.
3-6 months to see results.
but compounds over time.

## HOW GOOGLE WORKS

1. crawls ur site
2. indexes pages
3. ranks based on:
   - relevance (does it match the query?)
   - authority (do others trust u?)
   - experience (is the site good?)

## KEYWORD RESEARCH

find what people search:
1. brainstorm topics
2. use tools (Ahrefs, Ubersuggest, Google)
3. check search volume
4. check competition

target:
- high intent ("buy running shoes")
- low competition (long-tail)

## ON-PAGE SEO

### Title tag
```html
<title>How to Start a SaaS Company in 2024 | YourBrand</title>
```
- include keyword
- under 60 chars
- compelling

### Meta description
```html
<meta name="description" content="Learn how to start a SaaS..." />
```
- include keyword
- under 155 chars
- sells the click

### Headers
- H1: main topic (once)
- H2: sections
- H3: subsections
- include keywords naturally

### Content
- answer the search intent
- comprehensive
- well-structured
- original

## TECHNICAL SEO

basics:
- fast page load (<3s)
- mobile friendly
- HTTPS
- sitemap.xml
- robots.txt
- no broken links

## LINK BUILDING

authority comes from:
- other sites linking to u
- quality > quantity
- relevant sites in ur niche

how to get links:
- create linkable content (data, tools)
- guest posts
- PR / news
- directories

## COMMON MISTAKES

- keyword stuffing
- duplicate content
- ignoring mobile
- no internal links
- expecting instant results

## TOOLS

- Google Search Console (free, essential)
- Ahrefs / SEMrush (paid, powerful)
- Screaming Frog (technical audit)
- PageSpeed Insights (performance)

## QUICK WINS

1. add keywords to titles
2. fix broken links
3. improve page speed
4. add internal links
5. update old content
